Интерфейс QReal:BP

 
Общий вид интерфейса QReal:BP приведен на рисунке:

  1. Главное меню содержит набор базовых операций и настроек среды.
    • Пункты меню "Файл"
      • Открыть... — открыть сохраненный проект. Проекты хранятся на диске в файлах с расширением *.qrs.
      • Недавние проекты — открыть один из недавно использовавшихся проектов.
      • Сохранить — сохранить текущий проект.
      • Сохранить как... — сохранить текущий проект в выбранное местоположение. Последующие сохранения будут производиться в выбранный файл.
      • Сохранить диаграмму как картинку... — сохранить на диск текущее изображение сцены.
      • Выход
    • Пункты меню "Правка":
      • Отменить — отменить последнее действие.
      • Повторить — возобновляет отменённое действие.
      • Найти — открыть диалог поиска по проекту.
    • Пункты меню "Вид":
      • Приблизить — увеличить масштаб сцены.
      • Отдалить — уменьшить масштаб сцены.
      • Показывать сплешскрин — включить/выключить отображение стартового экрана с логотипом QReal.
    • Пункты меню "Инструменты":
      • Жесты мышью — открыть вкладку со списком доступных жестов мышью.
      • Проверить диаграммы — проверить все открытые диаграммы на корректность (подробнее см. Верификация бизнес-процессов).
      • Проверить текущую диаграмму — проверить текущую диаграмму.
    • Пункты меню "Настройки" (подробнее см. раздел Диалог настроек среды QReal:BP):
      • Настройки... — открыть окно настроек среды QReal:BP
      • Показать сетку — включить/отключить отображение сетки на сцене.
      • Показать направляющие — включить/отключить режим отображения вертикальных и горизонтальных направляющих линий для позиционирования объектов относительно друг друга.
      • Включить сетку — включить/отключить режим автоматического выравнивания по сетке.
      • Включить направляющие — осуществлять/не осуществлять выравнивание элементов по направляющим.
    • Пункты меню "Справка":
      • Помощь — открыть справочную систему QReal:BP.
      • О программе — просмотреть краткую информацию о QReal:BP (версия, сайт).
      • О Qt... — просмотреть информацию об инструментарии Qt, использованном для создания QReal:BP.
  2. Панель "Файл" повторяет основные операции, доступные из меню "Файл".
  3. Панель отмены и повтора действий повторяет соответствующие пункты меню "Правка".
  4. Панель "Вид":
    • Кнопки, позволяющие масштабировать диаграммы. Этого же можно добиться вращением колесика мыши, наведя курсор на произвольное место рабочей области диаграммы.
    • Кнопка, позволяющая перевести QReal:BP в "полноэкранный режим" — в этом режиме отображается только окно редактора и палитра элементов. Данный режим полезен для использования на экранах с небольшим разрешением. Этого же можно добиться, нажав комбинацию клавиш Ctrl + Shift + F.
  5. Панель "Верификация" содержит кнопки, позволяющие проверить на корректность текущую открытую диаграмму или все диаграммы проекта (подробнее см. раздел Верификация бизнес-процессов).
  6. Окно редактора отображает текущую диаграмму и позволяет ее редактировать.
  7. Миникарта — уменьшенный вид на диаграмму. Область, отображаемая в данный момент в окне редактора, обозначена на миникарте прямоугольником. Если перетащить его мышью в другое место миникарты, в основном окне будет видна соответствующая часть диаграммы.
  8. Палитра элементов содержит набор доступных элементов и связей между ними, которые можно добавить на диаграмму (подробнее о процессе создания диаграмм в QReal:BP см. раздел Создание моделей бизнес-процессов).
  9. Обозреватель диаграмм представляет диаграммы и элементы на них в иерархическом виде.
  10. Редактор свойств используется для отображения и редактирования значений свойств выделенных элементов.

Диалог настроек QReal:BP

Окно настроек QReal:BP открывается выбором соответствующего пункта меню "Настройки". Интерфейс данного окна содержит в себе две панели — левую, отображающую список вкладок с настройками, и правую, отображающую поля для редактирования настроек текущей вкладки.

Вкладка "Поведение"



  • Переключение табов в палитре — переключать/не переключать закладки в палитре элементов при смене редактора.
  • Автосохранение — задаёт частоту автосохранения в секундах.

Вкладка "Разное"



  • Антиалиасинг — включить/выключить режим сглаживания фигур при отрисовке.
  • Рендеринг OpenGL — использовать/не использовать OpenGL для рисования фигур (на некотором оборудовании ускоряет отрисовку элементов).
  • Показывать сплешскрин — показывать/не показывать заставку при старте QReal:BP.
  • Длина списка недавних проектов — позволяет задать длину списка недавних проектов, отображаемого в начальном меню и в меню Файл->Недавние проекты.
  • Рисовать старую линию — показывать/не показывать старое положение связи на диаграмме при ее деформации, а также, цвет которым будет эта старая связь рисоваться.

Вкладка "Редактор"



  • Показывать сетку — отображать/не отображать выравнивающую сетку на сцене редактора диаграмм.
  • Выравнивание по направляющим — осуществлять/не осуществлять выравнивание элементов по направляющим.
  • Показать направляющие — отображать/не отображать направляющие линии на сцене редактора диаграмм. Направляющие линии появляются, когда элемент находится на одной горизонтали или вертикали с другим элементом, и помогают выравнивать элементы на сцене.
  • Выравнивание по сетке — осуществлять/не осуществлять выравнивание элементов по сетке.
  • Толщина сетки — толщина линий сетки. Настраивается в зависимости от яркости монитора или проектора.
  • Размер — размер встроенного линкера, т.е. кружка над элементом на диаграмме, осуществляющего создание связей между элементами (см. раздел Создание элементов и связей).
  • Отступ — отступ встроенного линкера от пиктограммы элемента.
  • Режим рисования линии — позволяет выбрать из трёх режимов рисования линий: ломаная, квадратная и кривая линии. В режиме сквадрачивания линий связи между элементами прокладываются на диаграмме автоматически, образуя в точках излома прямые углы (подробнее о создании связей см. Создание элементов и связей).
  • Связи-петли — при создании связей с одинаковым началом и концом задаёт, насколько близко к элементу расположена связь.
  • Коэффициент масштабирования — коэффициент масштабирования диаграмм при увеличении/уменьшении с помощью колесика мыши или соответствующих операций панели "Вид".
  • Коэффициент масштабирования — коэффициент масштабирования диаграмм при увеличении/уменьшении с помощью колесика мыши или соответствующих операций панели "Вид".
  • Использовать системный шрифт — выбрать для отображения текста на сцене один из шрифтов операционной системы.
  • Представление — выбор между режимом отображения иконок и названий или только иконок в палитре.
  • Количество иконок в строке — количество иконок на строку палитры при выбранном режиме "Иконки".
